# mesto-react
Фронтенд проекта Mesto на React.

## Описание проекта
Цель этого проекта заключалась в отработке навыков программирования на курсе "Веб-Разработчик" от Яндекс Практикума. Особенность состоит в том, что сайт полностью был написан с использованием библиотеки React. На сайте также присутствуют возможность регистрации и авторизации.


## Языки разметки, которые присутствуют:

- HTML
- CSS

## Технологии, которые присутствуют на сайте:

- Flexbox вёрстка
- Файловая структура проекта по правилам Nested БЭМ
- Javascript
- Объекты
- Классы
- Модули
- Webpack
- Запросы на сервер
- React
- Регистрация
- Авторизация
